# 🍽️ Janoshi Restaurant Dashboard

A modern and responsive **Restaurant Management Dashboard** built with **React.js**, **Vite**, and **Tailwind CSS**. It helps restaurant owners and staff efficiently manage daily operations through an intuitive and user-friendly interface.

## ✨ Features

* 📊 Interactive Dashboard
* 🛒 Order Management
* 🍽️ Reservation Management
* 📋 Menu Management
* 🎁 Offers & Discounts
* 📦 Inventory Tracking
* 👨‍🍳 Staff Management
* 👥 Customer Management
* 💳 Payment Status Tracking
* 🔍 Search, Filter & Pagination
* 📱 Fully Responsive UI

## 🛠️ Tech Stack

* React.js
* Vite
* Tailwind CSS
* React Router
* React Icons
* Context API

## 🚀 Installation

```bash
# Clone the repository
git clone https://github.com/your-username/janoshi-restaurant-dashboard.git

# Navigate to project
cd janoshi-restaurant-dashboard

# Install dependencies
npm install

# Start development server
npm run dev
```
